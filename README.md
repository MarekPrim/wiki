# Installation

~~Pour obtenir les informations, envoyez 2 BTC sur 0xF204JF04JS083HD912HS04~~

Pour réaliser l'installation des librairies du projet :

```sh
$ cd frontend
$ npm i
$ cd ../backend
$ npm i
```

Pour lancer le projet :

```sh
cd backend
nodemon app.cjs &
cd ../frontend
npm run dev
```
